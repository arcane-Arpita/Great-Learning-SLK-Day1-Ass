What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It is known for its platform independence, meaning that Java programs can run on any device or operating system that has a Java Virtual Machine (JVM) installed.
What are the main features of Java?

Java has several key features including:
Object-oriented: Java is based on object-oriented programming principles such as inheritance, encapsulation, and polymorphism.
Platform independence: Java programs can run on any device or operating system with a Java Virtual Machine (JVM).
